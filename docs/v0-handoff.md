# V0 デザインハンドオフドキュメント

## プロジェクト概要

**プロダクト名**: Claude Code Agent Team
**コンセプト**: 「ながら作業を解放する、妖精エージェントチーム」

AI開発エージェントチームのランディングページ。6人の妖精モチーフの美少女キャラクターがチームとなり、ユーザーのコーディングを支援する世界観。メインエージェントが司令塔、5人のサブエージェントがそれぞれの専門分野を担当する。

**キーメッセージ**: 「コードに集中したまま、AIがあなたの耳に進捗を届ける。」

---

## 世界観・ビジュアルコンセプト

### テーマ
- **妖精（Fairy）× AI テクノロジー**
- 深夜のプログラミング中に妖精たちが現れて助けてくれるイメージ
- ファンタジーRPGのパーティ編成をモチーフにしたキャラクター紹介

### ムード・雰囲気
- 深夜の静けさと幻想的な光
- ダークUI × ネオングロウ × パーティクルエフェクト
- ゲームのキャラクター紹介画面のようなワクワク感
- 妖精の光の粒やキラキラが画面上を漂う

## やりたい機能・表現したいこと

### 1. ヒーローセクション
- 全画面のファーストビュー
- 「Fairy AI Agent Team」「Claude Code Agent Team」のタイトル表示
- キャッチコピー「コードに集中したまま、AIがあなたの耳に進捗を届ける。」
- サブエージェント5人のアイコンサムネイル横並び
- CTAボタン2つ：「声でレポートを聞く」（主）と「チームを見る」（副）
- 妖精のパーティクルエフェクト、キラキラの装飾

### 2. 音声フィードバック機能訴求セクション
- **コンセプト**: 「Eyes-Free Development」—— 声で聞く、目はコードへ。
- 左にコードエディタ風のビジュアル（ユーザーがコーディング中のイメージ）
- 右にキャラクターの音声バブル（波形アニメーション付き）
- 「コードを書いているあなた → キャラが音声で報告」というストーリーを視覚的に伝える
- 3つの特徴カード：「キャラ別の声」「目はコードだけに」「リアルタイム進捗報告」

### 3. キャラクターギャラリーセクション（メインセクション）
- **ガチャゲームのキャラクター紹介画面**のようなカード表現
- メインエージェント（光）は特別仕様のフル幅カード
- サブエージェント5人はグリッド配置
- 各カードに以下を含む：
  - キャラクターの立ち絵画像
  - 名前・担当・キャッチフレーズ
  - RPGスタイルのスキルバー（3つ）
  - 性格タグバッジ
  - ボイスサンプル再生ボタン
- キャラクターごとのテーマカラーでカードが色づく
- ホバーでグロウエフェクト、スキルバーはスクロール時にアニメーション

### 4. 仕組み説明セクション（How It Works）
- 4ステップで動作フローを説明：
  1. **タスク受付** — ユーザーからの依頼をメインエージェントが受け取り分析
  2. **タスク振り分け** — タスク内容に応じて最適なスペシャリストを選定・指示
  3. **実行・統合** — 各スペシャリストが実行、メインエージェントが成果を統合
  4. **音声で報告** — 完了と進捗をキャラクターの声でリアルタイムにお知らせ

### 5. チーム連携ダイアグラムセクション
- メインエージェント（光）を中心に、5人のサブエージェントが放射状に配置された連携図
- 各エージェント間の繋がりを視覚的に表現
- 誰がどの役割を担っているかが一目で分かるダイアグラム

### 6. フッター
- プロダクト名とセクションへのナビリンク

## 全体で大事にしたいポイント

1. **キャラクターの魅力を最大限引き出す** — ゲーム的なカード演出でキャラの個性（スキル・性格・セリフ）を視覚的に伝える
2. **音声フィードバック機能の訴求** — 「コーディング中に画面を見なくてもAIの進捗が耳で分かる」という独自価値を前面に
3. **ファンタジー×テクノロジーの世界観** — 深夜のプログラミングに妖精が現れる幻想的な雰囲気
4. **チーム連携の可視化** — メインエージェントを中心にサブエージェントが有機的に連携する構図
5. **モバイルファースト** — スマホから美しく表示。段階的にデスクトップで情報量を増やす
6. **インタラクティブな演出** — パーティクル、グロウ、スキルバーアニメーションなどで没入感を出す
